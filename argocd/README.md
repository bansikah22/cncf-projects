# Argo CD Demo

This document provides a verifiable demo of Argo CD's basic functionality.

## Demo Overview

This demo will:
1.  Install the Argo CD CLI.
2.  Install Argo CD on a local Minikube cluster.
3.  Verify the Argo CD installation.
4.  Deploy a sample application using Argo CD.
5.  Verify the application deployment.
